Embed on Another Website

Drop the signup form into any website using a single script tag. The iframe auto-resizes to fit its content and posts a message event on successful signup.

Method 1 — JS Snippet Recommended

Add a placeholder div where you want the form, then include the script once. Auto-resizes, handles all styling.

Success Event Hook (optional)

      
The embed fires a custom event on the parent page when a form is submitted successfully — useful for analytics, redirects, or showing a custom thank-you.

      
window.addEventListener('signflow:success', function() {
  // e.g. gtag('event', 'signup'); or redirect
  console.log('Form submitted successfully!');
});

⚠️ The secret key is stored in config.json on your server. Never share it or commit it to a public repo.

      

      
    

    

      
🔐 Authentication (Auth0)

      

        Admin access is protected by Auth0. Only users with the signflow-admin role in your Auth0 tenant can log in.


        To manage users and roles, go to your Auth0 Dashboard → User Management → Roles.
      

      

        
        
      

    

    

      
GDPR Compliance

      

        ✅ Consent recorded with timestamp and IP on every subscription

        ✅ Unsubscribe links generated per subscriber

        ✅ Data deletion endpoint available (/delete-data)

        ✅ Cookie consent banner on public page

        ✅ Privacy policy page at /privacy

        ✅ Admin can delete individual records

        ✅ Export to CSV/JSON for data portability
      

    

    

      
Deployment Info

      

        Data is stored in data/subscribers.json and data/config.json.

        Uploaded images are in public/uploads/.

        Run npm start on your Lightsail instance. Use PM2 for persistence:

        npm install -g pm2 && pm2 start server.js --name signflow && pm2 save
